# 요구 사항

## Header

- 한달 수입 확인
- 다음달 결제 금액 확인
  - 신용카드, 버스 등 
- 고정 지출 금액 확인
  - 통신료, 보험, 대출 등
- Flex 분류 (제일 많이 지출한 분류)

## Body

- 지출 내역
    - 리스트 형식
    - 은행별 -> 금액별로 정렬
      - xx 은행
        - 지출 1
        - 지출 2
        - ...
- 입금 내역
    - 리스트 형식
    - 지출 내역과 동일한 형식 사용
- 통계
  - 금액 (바 차트)
    - 바 차트 형태
    - \[ 시간, 분류 \]로 선택해서 확인할 수 있게함
  - 분류 ( 도넛 )
    - 도넛 차트 형태
    - 분류별로 얼마나 사용했는지 Count 확인
- 간단한 메모
